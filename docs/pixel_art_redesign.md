# 像素风战场与时代美术说明

本轮运行时美术已经整理到 `Assets/Resources`，旧素材包和生成中间目录已删除。游戏不会直接加载根目录 `素材包` 或 `Assets/Art/Generated`。

## 当前使用资源

- 时代地图：`Assets/Resources/Battle/Maps/PixelFrontline_*.png`
- 通用设施：`Assets/Resources/Battle/Facilities/`
- 时代基地：`Assets/Resources/<Era>/Base/Base.png`
- 时代兵种：`Assets/Resources/<Era>/Units/<Unit>/move_01..05.png` 与 `attack_01..05.png`
- 时代炮塔：`Assets/Resources/<Era>/Towers/<Tower>/attack_01..05.png`

## 内容规模

- 5 个时代：蛮荒部落、机械工坊、电力时代、核能纪元、星海文明
- 每个时代 5 个兵种
- 每个时代 3 个炮塔
- 每个时代 1 张同布局、不同风格的战场地图

## 清理说明

以下目录属于旧素材包或生成中间产物，已从项目中删除：

- `素材包`
- `Assets/Art/Generated`
