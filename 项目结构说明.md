# 战线进化：纪元之战 - 项目结构说明

本文档用于说明当前仓库的目录职责。新的成员如果只想快速运行项目，先看 `README.md`；如果要增补资源或代码，再参考这里。

## 根目录

```text
War-of-Eras/
  Assets/            Unity 游戏内容
  Packages/          Unity 包依赖声明
  ProjectSettings/   Unity 项目设置
  README.md          项目总览、运行方式和玩法说明
  开发框架说明.md    玩法代码与扩展规则
  项目结构说明.md    当前目录职责说明
```

## Assets 目录

```text
Assets/
  Art/                 美术草图、候选图和非运行时资源
  Audio/               旧音频素材或外部音频素材
  Data/                后续 ScriptableObject 数据化的预留目录
  Editor/              Unity 编辑器辅助工具
  Prefabs/             后续 prefab 化内容的预留目录
  Rendering/           URP 渲染资源与后处理配置
  Resources/           当前运行时通过 Resources.Load 加载的实际资源
  Scenes/              MainMenu、Battle 等场景
  Scripts/             项目脚本
  Settings/            Unity 设置资源
  TextMesh Pro/        TextMesh Pro 默认资源
```

## Scripts 目录

```text
Assets/Scripts/
  Battle/
    Core/
      BattleGameController.cs          战斗主控制器
      BattleGameController.Sprites.cs  程序化 UI、标记和默认 sprite
      BattleGameController.Types.cs    战斗控制器内部辅助类型
      BattleLayout.cs                  可选场景布局标记读取
      BattleRuntimeActors.cs           单位、防御塔、资源点等运行时对象
      GameSetup.cs                     主菜单和战斗共享的地图/难度配置
    UI/
      .gitkeep                         预留 UI 脚本目录
  MainMenu/
      MainMenuController.cs            主菜单运行时 UI 和页面切换
      MainMenuController.Helpers.cs    主菜单 sprite、字体、hover 效果
```

目前核心玩法集中在 `Battle/Core` 中。后续如果模块继续变大，可以再拆成 `Combat`、`Economy`、`Progression`、`AI`、`UI` 等目录。

## Resources 目录

当前项目主要依赖 `Assets/Resources`，脚本通过字符串路径加载资源。

```text
Assets/Resources/
  Audio/Ambience/      战场底噪和时代环境音
  Battle/Facilities/   资源点等通用设施贴图
  Battle/Maps/         五个时代的战场地图
  Battle/Outcome/      胜利/失败结算背景
  MainMenu/            主菜单背景和标题
  Barbarian/           蛮荒部落资源
  Machine/             机械工坊资源
  Electric/            电力时代资源
  Nuclear/             核能纪元资源
  Starsea/             星海文明资源
```

每个时代目录通常包含 `Base`、`Units`、`Towers` 三类内容。新增资源时要保持当前命名风格，否则 `Resources.Load` 会找不到文件。

## Scenes 目录

- `MainMenu.unity`：主菜单入口，挂载 `MainMenuController`。
- `Battle.unity`：正式战斗场景，挂载 `BattleGameController`。
- `BattleSettings.lighting`：战斗场景的光照设置。

## Editor 目录

- `AutoPlayFromMainMenu.cs`：编辑器辅助入口，用于统一从主菜单开始播放。
- `BattleArtImportSettings.cs`：批量修正蛮荒时代图片导入设置的工具。

## 协作建议

- 改玩法逻辑前，先确认变更属于战斗主控制器、运行时对象、菜单还是资源加载。
- 新增时代资源时，优先放在对应时代的 `Resources/时代目录` 下。
- 新增规则数据时，当前先在 `BattleGameController.BuildDefinitions()` 接入；后续可迁移到 `Assets/Data` 的 ScriptableObject。
- 删除脚本前先查场景和 prefab 是否引用对应 `.meta` 的 GUID，避免 Unity 出现 Missing Script。
